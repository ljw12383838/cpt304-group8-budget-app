# Budget App - CPT304 Group 8 Baseline

This repository contains the baseline version of the Personal Budget App selected for CPT304 Coursework 1.

This initial `main` branch is intentionally kept close to the original application. Enhancements will be added through separate feature branches and Pull Requests so that each team member's contribution is visible in GitHub.

## Planned Enhancement Branches

- `feature/security-xss`: secure rendering for user-entered budget entries.
- `feature/validation-tests`: input validation, model logic, and unit tests.
- `feature/accessibility-i18n`: semantic accessibility improvements and English/Chinese UI support.
- `feature/privacy-storage`: localStorage consent banner, privacy policy, and data clearing controls.
- `feature/integration-ci-deployment`: CI, Codecov, README, Vercel deployment, and final integration checks.

## Baseline Usage

Open `index.html` in a browser to run the original budget app locally.

No npm setup is required in this baseline branch. The later enhanced version will add Vite, Vitest, coverage reporting, Codecov, and deployment configuration.

## Contribution Rule

Do not push directly to `main`. Create a feature branch, commit your assigned module, and open a Pull Request.
